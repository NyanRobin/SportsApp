# 🔐 관리자 권한 요청 가이드

## 현재 상황
- **사용자 계정**: ryanchoi
- **권한 상태**: 일반 사용자 (관리자 권한 없음)
- **관리자 그룹**: root만 포함됨

## 📋 관리자 권한 요청 방법

### 방법 1: 시스템 관리자에게 직접 요청 (가장 일반적)

**요청 내용**:
```
안녕하세요,

iOS 앱 개발을 위해 다음 작업이 필요합니다:

1. 관리자 권한 부여 요청
2. 또는 다음 명령어 실행 요청:
   sudo gem install cocoapods

목적: Flutter iOS 앱 빌드 및 App Store 배포
소요 시간: 5-10분
```

### 방법 2: IT 부서/시스템 관리자에게 이메일

**이메일 템플릿**:
```
제목: [긴급] iOS 앱 개발을 위한 관리자 권한 요청

안녕하세요,

현재 Flutter로 개발한 iOS 앱을 App Store에 배포하려고 하는데, 
CocoaPods 설치를 위해 관리자 권한이 필요합니다.

요청 사항:
1. ryanchoi 계정에 sudo 권한 부여
2. 또는 다음 명령어를 대신 실행해주시기 바랍니다:
   sudo gem install cocoapods

목적: iOS 앱 빌드 및 배포
소요 시간: 5-10분
영향도: 시스템에 부정적 영향 없음 (개발 도구 설치만)

감사합니다.
```

### 방법 3: 회사 정책 확인 후 요청

**확인할 사항**:
- 개발자 도구 설치 정책
- 관리자 권한 부여 절차
- 임시 권한 부여 가능 여부

### 방법 4: 대안 방법 제시

**관리자에게 제시할 대안**:
```
관리자 권한이 어려우시다면, 다음 방법들도 가능합니다:

1. Apple Developer Program 가입 후 Xcode에서 직접 빌드
2. 다른 개발 머신에서 빌드
3. 임시 관리자 계정 제공
```

## 🔄 권한 부여 후 작업 순서

```bash
# 1. CocoaPods 설치
sudo gem install cocoapods

# 2. iOS 의존성 설치
cd ios
pod install
cd ..

# 3. iOS 빌드 테스트
flutter build ios --release --no-codesign

# 4. Apple Developer Program 가입 후 최종 빌드
```

## 🚨 대안 방법 (권한 요청이 어려운 경우)

### 1. Apple Developer Program 가입 ($99/년)
- Xcode에서 직접 빌드
- CocoaPods 문제 우회
- 가장 확실한 방법

### 2. 다른 Mac 사용
- 관리자 권한이 있는 다른 컴퓨터
- 프로젝트 복사 후 빌드

### 3. 클라우드 개발 환경
- GitHub Codespaces
- GitPod
- 온라인 Flutter 개발 환경
